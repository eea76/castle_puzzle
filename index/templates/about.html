This puzzle originally appeared at the end of Square Enix's 1999 game Final Fantasy VIII, and I was inspired to try recreating it as a simple web application. I'm pleased with how well this escape room-style puzzle translated to a point-and-click interface.

I obtained all the assets using the screen-capture feature on a Nintendo Switch, transferred them to my computer where I edited out the character sprites and stitched the various pieces of the rooms together in Photoshop. Then I used Python's Django framework to create the programming logic and web application. In total, the project took around 15 hours to complete, not counting the time spent swearing at JavaScript.

This app keeps track of your progress as you play, which is why it's necessary to create an account beforehand. Otherwise it would not be possible to track the progress of a given player, and only one person could play at a time (and annoyingly, I would have to manually reset the app each time).

The codebase is available as a public repository on Github for anyone interested in the technical details. To report any problems, suggestions, or to learn more about how I built this, you can email me at elon.arbiture@gmail.com. Thanks.
